Wave Survival Changelog

Release 1.3
January 28, 2020
112 code commits (1,159 total)

This release focused on adding new wave types to make the gameplay more varied and fun.

  • AddedNew map: Van Horn Trading Post
  • AddedMore tips for new players
  • AddedArmor! You can now find an armor spawn on each map. Each level of armor gives you more resistance against enemies.
  • AddedWAVE TYPES! Waves can now be more varied, like single-weapon waves, vampires and wolves, robots, bosses, and more!
  • AddedWaves can now be either quota based (kill number of dudes) or survive for a time
  • AddedNew powerup: teleport! This allows you teleport a short distance, allowing for quick repositioning
  • ChangedIncreased price of Rolling Block to $9.50
  • ChangedMade powerups visible from afar
  • ChangedChat now fades out if no messages were received recently
  • ChangedUpdated server version to latest stable branch
  • FixedDead bodies nast better than before
  • FixedShop now works! Feel free to earn money ingame then go buy some skins!
  • FixedFixed ingame changelog not updating.
  • FixedFixed a bug that stopped players from entering games sometimes
  • FixedFixed a typo in chat message when a player leaves
Release 1.2
January 5, 2020
203 code commits (1,047 total)

This release focused on adding new features to make the gamemode more enjoyable.

  • AddedNew map: Rhodes
  • AddedPowerups! Enemies now can drop powerups when killed. Powerups give players temporary boosts and abilities, such as double money, instant kill, lightning strike, and more
  • AddedPlayer leave/join messages in chat
  • AddedWeapon spawns and powerup spawns now have glowing zones around them to make them more visible
  • AddedShop! The shop is now active. However, there are only a few character models to choose because not all models work properly, and it requires manual testing. We are investigating how to make all models work, so please hold on! All money earned ingame is added to your shop money
  • AddedBuying a weapon at a weapon spawn now restores the durability/condition of the weapon
  • AddedUpdates section to the main menu. Now you can see all the updates while you are on the server.
  • ChangedStopped loading all parts of maps, and instead now only load in certain parts of maps. This means that the streets are no longer cluttered, and the missing green house in Valentine still spawns. Less lag!
  • ChangedComplete rewrite of our enemy spawning system. This will allow us smoothily implement different kinds of enemies. (different wave types coming soon woo!)
  • FixedPlayers not being able to start games sometimes
  • FixedHopefully fixed an issue where enemies are invisible
  • FixedWeapon and revive icons not showing up sometimes
  • FixedPlayers not reviving properly on new rounds, which caused them to be invincible and games to never end
  • FixedDead bodies not being removed properly. It mostly works now, at least better than before
