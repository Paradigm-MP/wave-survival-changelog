Survival Island Changelog

1.1 Release Update October 17, 2018

Inventory
  • AddedAdded an animation to item durabilities when changed
  • FixedUpdated responsiveness of inventory tooltips
  • FixedUpdated responsiveness of item usage, shifting, and rearrangement
  • FixedFixed various small issues
  • FixedFixed not being able to drop one item out of some stacks, such as Googly Eyes (thanks Hugom!)
  • FixedFixed a bug where the inventory would not load for some players due to custom properties
  • FixedFixed a bug where some items would not unequip when dropped
  • ChangedDeath drops have been reworked. You no longer randomly drop entire stacks from your inventory when you die base on level. Instead, certain stacks are randomly chosen and a certain percentage of items from those stacks are dropped. This still does allow for the entire stack to be dropped, but it does not happen all the time. This has been worked into the new levelling system.
Loot
  • AddedAdded Superboxes. Sometimes a normal lootbox will become a Superbox, which means that it will have loot from all tiers, including legendary tier items.
  • AddedAdded loot hotzones. Certain items can only be found in these hotzones.
  • AddedAdded sound effects to loot.
  • AddedAdded 300+ more loot spawns. Total is now OVER 9100.
  • FixedFixed an issue where lootboxes that were very far would sometimes load and cause slight frame drops
Items
  • AddedAdded Model Change
    • This item allows you to change into other character models besides Rico
  • AddedAdded Landclaim
    • Comes in sizes ranging from 50m to 400m, though larger sizes are rarer
    • This item allows you to claim an area of land so that you can build in it
  • AddedAdded Storage
    • Replaces Small and Large Storages
    • By default, it can hold up to 6 items
    • Can no longer be sometimes placed underwater
    • You can rename it
    • You can no longer dismount it from your storages menu on F7. Instead, you have to open it up and then rename it to DISMOUNT
    • You can apply various upgrades to improve it
      • Space Upgrade
        • For each applied, adds +1 space to a Storage
        • Can have up to 6 applied to a Storage
      • Keypad Lock
        • Use to lock your Storage with a 4 digit combination
      • Identity Lock
        • Use to lock your Storage to Only Me, Friends, or Everyone
      • Explosive Upgrade
        • Causes the Storage to explode and kill the person who broke into it if someone uses a Hacking Module to open it, also destroying all the items inside along with the storage
        • Can be disarmed using a Disarm Module
      • Zap Upgrade
        • Causes the storage to zap and kill the person who broke into it if someone uses a Hacking Module to open it
        • Zap Upgrade keeps working until removed or disarmed
        • Can be disarmed using a Disarm Module
      • Shield Upgrade
        • Blocks one hacking attempt and then is destroyed
        • Can have a maximum of 4 on one storage
      • Disguise Upgrade
        • Causes the Storage to appear as something other than a storage
        • Possible disguises: tree, rock, explosive barrel
  • AddedAdded Hacking Module
    • This item can be used to hack into storages
  • AddedAdded Disarming Module
    • This item can be used to disarm traps that are on storages
  • AddedAdded Trap Detector
    • This item can be used to scan a storage for traps
